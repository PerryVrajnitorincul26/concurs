Server

Serverul se bazeaza pe libraria BaseHTTPRequestHandler clasa folosita este o sublcasa a unei clase din acest modul si este scris in python

Metode implementate

do_head({request})

Aceasta metoda incearca sa deschida fisierul mentionat in request si in cazul in care acesta exista returneaza codul 200
In caz contar returneaza 404
O metoda de verificare similara are loc si la metoda do_get()

do_get({request})

Incearca sa deschida fisierul precizat si in cazul in care acesta exista il raspune la request cu fisierul cerut in format text/"UTF-8"

Fisiere suportate
Text .js .css .html
Imagine .png .jpg

In cazul in extensia nu este suportata serverul va returna codul 415 cu mesajul "415 unsupported media extension"
